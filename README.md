# Cube Stack -- 3D Interactive Math Education Tool

An interactive 3D cube stacking tool for math education. Place and visualize unit cubes in 3D space to explore spatial reasoning concepts.

**[Live Demo](https://masonthink.github.io/cube-stack/)** | [About](https://masonthink.github.io/cube-stack/about.html)

## Features

- Place / remove cubes by entering XYZ coordinates (0-9)
- 6 preset shapes: Cube, Pyramid, Stairs, Tower, L-Shape, Cross
- Real-time 3D rendering with auto-colored layers
- Ghost preview, click-to-select, auto-fill support blocks
- Toggleable axes, grid labels, and auto-rotation
- Live stats: total cubes, layers, coordinate range

## Tech Stack

- [Three.js](https://threejs.org/) v0.170 (CDN)
- Vanilla JavaScript, single-file deployment
- GitHub Pages hosting

## Usage

Open `index.html` in any modern browser. No installation required.

Or visit the [live demo](https://masonthink.github.io/cube-stack/).

# 立方体堆叠 -- 3D 交互式数学教学工具

一个 3D 交互式数学教学工具，在三维空间中自由放置和可视化立方体。

**[在线演示](https://masonthink.github.io/cube-stack/)** | [功能说明](https://masonthink.github.io/cube-stack/about.html)

## 功能

- 输入 XYZ 坐标（0-9）放置 / 移除方块
- 6 种预设形状：正方体、金字塔、阶梯、塔、L 形、十字
- 实时 3D 渲染，按层自动着色
- 幽灵预览、点击选中、自动填充支撑
- 坐标轴、网格标号、自动旋转等开关
- 实时统计方块总数、层数、坐标范围

## 技术栈

- [Three.js](https://threejs.org/) v0.170（CDN 引入）
- 纯 Vanilla JavaScript，单文件部署
- GitHub Pages 托管

## 使用方式

在浏览器中打开 `index.html` 即可，无需安装。

或访问[在线演示](https://masonthink.github.io/cube-stack/)。
