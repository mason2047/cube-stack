# 立方体堆叠

一个 3D 交互式数学教学工具，在三维空间中自由放置和可视化立方体。

**[在线演示](https://mason2047.github.io/cube-stack/)**  |  [功能说明](https://mason2047.github.io/cube-stack/about.html)

## 功能

- 输入 XYZ 坐标（0-9）放置 / 移除方块
- 6 种预设形状：正方体、金字塔、阶梯、塔、L 形、十字
- 实时 3D 渲染，按层自动着色
- 幽灵预览、点击选中、自动填充支撑
- 坐标轴、网格标号、自动旋转等开关
- 实时统计方块总数、层数、坐标范围

## 技术栈

- Three.js v0.170（CDN 引入）
- 纯 Vanilla JavaScript，单文件部署
- GitHub Pages 托管
